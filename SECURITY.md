# Security Policy

## Supported Versions

| Version | Supported |
| ------- | ---------- |
| 11.x    | :white_check_mark: |
| 10.x    | :white_check_mark: |
| 9.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please send an email to hello.woop@uncoverthefuture.org. All security vulnerabilities will be promptly addressed.

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Proposed fix (optional)

## Response Timeline

We aim to acknowledge reports within 48 hours and provide a fix timeline within 7 days.
